To compare categories, use the checkboxes on the top of the table to select more than one category and then press the "Compare categories" button. All other categories will be hidden and numerical data will be color coded relative to the other data in that category.
To evaluate options, use the checkboxes on the top of the table to select a single category and then press the "Evaluate options" button. Upon doing so, a dialog box will appear asking you to choose a unique item within the selected category. Once selected, all entries with that item in that category will be isolated, and if there's more than one entry, the numerical data will be color coded relative to the other data in that category.
